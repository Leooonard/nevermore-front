    消息状态，目前支持 TO_SENT（待发送）| SENT_SUCCESS（发送成功）| SENT_FAILED（发送失败）| CANCELED（已取消）；为空表示获取所有状态

    开始日期，格式为"YYYY-MM-DD"；为空表示无开始时间

  • endDate: (date )

    结束日期，格式为"YYYY-MM-DD"；为空表示无结束时间

  • pageSize: (integer )

    默认为10

  • pageNum: (integer )
